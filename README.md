# cs_371
Five homeworks made with the Solar2D mobile app SDK for FA2025 CS 371: Mobile Computing App Inct &amp; Development at UAH

# Info:
- The Solar2D APIs are implemented in Lua, and they are made use of via Lua as well
- For each of these assignments, my source code is contained in .lua files in the root folder
- These applications can be run by downloading the Cornoa Simulator (https://solar2d.com/#), clicking "Open Project" in the main menu of the simulator, and selecting the main.lua file in whatever application you wish to run
